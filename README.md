## How to use

- setUp `pnpm`

- install

  ```bash
  pnpm install
  ```

- build 

  ```bash
  pnpm run build
  ```

- start

  ```bash
  pnpm run dev
  ```

  Go to [http://localhost:3000/](http://localhost:3000/)
